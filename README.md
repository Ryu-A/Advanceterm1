# Atte 勤怠管理アプリ
社員の勤怠管理（記録と確認）を一括管理することができるアプリ

##作成した目的
物理的なタイムカードではなく、すべての社員の勤怠（出勤、退勤、休憩）をサーバー上で管理ができることを目的として作成。
